# Projetos em Power BI

Este repositório contém dashboards criados no Power BI para análise de dados.

## Projetos

- **Vendas Anuais** – Análise de vendas com filtros por região e produto.
- **Financeiro Pessoal** – Acompanhamento de despesas mensais e metas de economia.

## Como abrir

Abra os arquivos `.pbix` com o Power BI Desktop.
